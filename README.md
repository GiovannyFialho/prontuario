# Prontuario

## Instalação das dependências
```shell
yarn
```
## Iniciando modo de desenvolvimento
```shell
yarn dev
```

## Testes
### Comando para rodar teste
```shell
yarn test
```
### Observando alterações nos testes
> É preciso ter dado pelo menos o ```git init``` no projeto antes de iniciar esse comando.
```shell
test:watch
```
## Storybook
```shell
yarn storybook
```
